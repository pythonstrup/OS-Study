# Compile

## gcc

- installation

```
$ sudo apt-get install gcc
```

- compile and construct execute file(a.out)

```
$ gcc code.c
```

- only compile(construct only *.o file)

```
$ gcc -c code.c
```

- naming execute file

```
$ gcc -o naming.out code.c
```

<br/>

## make

- Installation

```
$ sudo apt install make
```

- Execute

```
$ make
$ ./add.out
```

- Clear

```
$ make clean
```
